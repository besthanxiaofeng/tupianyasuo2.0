# 图片压缩工具

这是一个简单易用的在线图片压缩工具，具有苹果风格的界面设计。

## 功能特点

- 支持上传 PNG、JPG 格式图片
- 实时预览原图和压缩后的图片
- 显示压缩前后的文件大小
- 可调节压缩比例
- 支持压缩后图片下载
- 响应式设计，支持各种设备

## 使用说明

1. 点击上传区域或将图片拖拽到上传区域
2. 查看原图和压缩后的预览
3. 使用滑块调节压缩质量(1-100%)
4. 点击下载按钮保存压缩后的图片
